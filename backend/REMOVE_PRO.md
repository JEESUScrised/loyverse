# Отключение Pro подписки

## Быстрый способ

Запустите скрипт:
```bash
cd backend
npm run remove-pro
```

Или напрямую:
```bash
cd backend
node remove-pro-subscription.js
```

## Что делает скрипт

1. Находит всех владельцев в базе данных
2. Устанавливает `plan = 'start'`
3. Устанавливает `subscriptionStatus = 'inactive'`
4. Удаляет `subscriptionEndDate`
5. Обновляет все подписки заведений на план 'start'

## Если владелец еще не создан

Владелец создается автоматически при первом входе в owner приложение. 
После создания владельца запустите скрипт снова.

## Проверка владельцев

Чтобы посмотреть всех владельцев в базе:
```bash
cd backend
node check-owners.js
```
